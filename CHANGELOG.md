## [1.0.0] - 2026-02-06

### Added
- Initial release of @kdrdesigns/lego Design System
- Complete component library with 12 components:
  - Layout: Box, Stack, Inline
  - Typography: Text
  - Interactive: Button, IconButton, Toggle
  - Forms: Input, Textarea
  - Content: Card, Badge, Modal
- TypeScript support with full type definitions
- CommonJS and ESM module formats
- Comprehensive documentation and examples
- WCAG AA accessibility compliance
- Design tokens and theme system
- Focus management and keyboard navigation
- Playful "blocks" aesthetic with chunky borders and soft shadows
